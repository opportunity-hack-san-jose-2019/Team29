Running Locally
Make sure you have Node.js and the Heroku Toolbelt installed.

git clone git@github.com:opportunity-hack-san-jose-2019/Team29.git # or clone your own fork
cd Team29
npm install
npm start
Your app should now be running on localhost:3000.
